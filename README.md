# Gym Landing Page 🏋️‍♂️

A modern, highly responsive landing page for a gym or fitness center. This project is built using the latest frontend technologies, featuring blazing-fast development with Vite, modern styling with Tailwind CSS v4, and seamless animations using GSAP.

## 🛠 Tools & Technologies Used

This project leverages the following modern web technologies:

* **Core:** [React 19](https://react.dev/) & [React DOM](https://react.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/) (for fast HMR and optimized builds)
* **Routing:** [React Router v7](https://reactrouter.com/)
* **Styling & UI:** * [Tailwind CSS v4](https://tailwindcss.com/) (via `@tailwindcss/vite`)
    * [DaisyUI v5](https://daisyui.com/) (Tailwind component library)
* **Animations:** [GSAP](https://gsap.com/) (GreenSock Animation Platform)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
* **Code Quality:** ESLint

## 🚀 Project Setup Instructions

Follow these steps to get the project up and running on your local machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine (version 18+ is recommended).

### 1. Clone the repository
```bash
git clone <your-repository-url>
cd gymlandingpage


2. Install dependencies : npm install
Using npm (or your preferred package manager like yarn, pnpm, or bun)




3. Start the development server : npm run dev


This will start the Vite development server. Open your browser and navigate to http://localhost:5173 (or the port specified in your terminal).
